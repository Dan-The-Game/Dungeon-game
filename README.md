# ASCII Dungeon Game

A simple ASCII art, turn-based dungeon game in Python.

## Run

From the project root:

```
python src/main.py
```

## Controls

- `W` `A` `S` `D` to move
- `Q` to quit
- `U` to use power up

Reach the exit `E` and avoid or defeat monsters `M` and `X`.
- `+` is health
- `P` is power up
